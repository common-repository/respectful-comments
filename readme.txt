The Respectful Comments plugin replaces the text on the comments submit button.

== Description ==

Respectful Comments is a Wordperss plugin that changes the text of the Submit Comment button to <b>My comment respects the blog and its readers</b>.

Readers that want to comment in your blog will:
<ol>
<li>Read your expectation from them in a one line, easy to understand text</li>
<li>Will not skip reading what you expect from them.</li>
<li>Will not need to click "I have read the termsâ€¦".</li>
</ol>
My comment respects the blog and its readers is the default text that this plugin comes with. You can change the text to anything you wish, in any language.

== Installation ==

1. Upload the `respectful-comments` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Am I limited to the text that comes with the plugin? =

No, go to Settings > Respectful Comments and change the text.

See the [languages page](http://info.org.il/respect/languages) at the plugin website for translations to the default text of the plugin.

= It doesn't work in my blog =

See the [technical page](http://info.org.il/respect/languages) at the plugin website for hints to what can you do.
